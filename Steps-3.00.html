Showing Duplicate Records and creating a CTE syntax .

 

In the table, duplicate records are circled in red. Example is David showing 3 times. We only want David to show one time. Same goes to other students in the table. 

Using Common Table Expression (CTE) , we created a new column called ROWID to partition the multiple data.  So, we can easily identify how many multiple rows in the table.
